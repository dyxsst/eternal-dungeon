# Eternal Dungeon - Infinite RPG

An infinite-floor, procedurally generated dungeon crawler RPG built with Phaser 3.

## Features
- Infinite procedurally generated dungeons
- 3 combat modes: Melee, Ranged, Magic
- 5 enemy types with scaling difficulty
- Full RPG progression system
- Loot system with 5 rarity tiers
- Cross-platform: PC and Mobile
- Persistent saves with InstantDB

## How to Play

### PC Controls
- **Movement**: WASD or Arrow Keys
- **Switch to Ranged**: R
- **Switch to Magic**: F
- **Inventory**: I
- **Interact**: E

### Mobile Controls
- Use on-screen D-pad for movement
- Tap action buttons to switch combat modes
- Tap inventory button to manage items
